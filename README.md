# 🧱 LEGO STORE 

Sistema CRUD desenvolvido em PHP para gerenciamento de produtos LEGO, utilizando phpMyAdmin para manipulação do banco de dados.

## 🚀 Funcionalidades

* Cadastro de produtos
* Edição de produtos
* Exclusão de produtos
* Listagem dinâmica
* Busca de produtos
* Imagens personalizadas
* Interface moderna e responsiva

## 🛠️ Tecnologias

* PHP
* MySQL
* phpMyAdmin
* HTML5
* CSS3
* JavaScript
* XAMPP

## 📦 Estrutura do Produto

Cada produto possui:

* Nome
* Quantidade de peças
* Preço
* Loja
* Imagem

## 🎯 Objetivo

Projeto criado para praticar operações CRUD (Create, Read, Update e Delete) e integração entre PHP e banco de dados MySQL.

## ▶️ Como executar

1. Clone o repositório
2. Coloque a pasta dentro do `htdocs` do XAMPP
3. Inicie Apache e MySQL
4. Importe o banco de dados no phpMyAdmin
5. Acesse pelo navegador:

```bash
http://localhost/bancoSimples
```
